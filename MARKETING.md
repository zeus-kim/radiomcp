## Full Description (for detailed listings)

### radiomcp — Internet Radio + AI DJ for Claude

Turn your AI assistant into a world-class radio DJ. radiomcp brings 55,000+ internet radio stations from 197 countries directly into Claude, with powerful AI DJ capabilities.

**Key Features:**

🎵 **55,000+ Radio Stations** — Jazz, K-pop, news, classical, and more from every corner of the globe

🎙️ **AI DJ Broadcast** — 24/7 personalized DJ sets with voice commentary in 10+ languages

🍎 **Apple Music Integration** — Native library playback via persistent ID, no GUI clicking required

📺 **Video Playback** — Watch YouTube and live streams in a desktop window

🔍 **Smart Search** — Find stations by mood, genre, country, or just describe what you want

⭐ **Favorites & History** — Save your favorite stations and track listening history

🎚️ **Full Playback Control** — Volume, stop, now playing, song recognition (Shazam-style)

**Just ask Claude:**
- "Play some late-night jazz"
- "Start a DJ set with Bruno Mars"
- "Show me YTN live news"
- "What's playing right now?"

**Quick Install:**
```bash
pip install radiomcp && radiomcp
```

Auto-registers with Claude Desktop on first run. No configuration needed.
